# ✅ My To Do App

A simple and clean To Do application built to manage daily tasks efficiently.

## 📸 Screenshot

![To Do App Screenshot](./image.png)

## 🚀 Features

- Add new tasks
- View task details
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Track task status (completed / not completed)

## 🛠️ Technologies Used

- Python
- Django
- HTML
- CSS
- SQLite
